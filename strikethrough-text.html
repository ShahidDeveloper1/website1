What Is Strikethrough Text?

Strikethrough text is text with a horizontal line drawn through the middle of each character. In Unicode, this effect is achieved by appending the combining long stroke overlay character (U+0336) after each letter. The result — like S̶t̶r̶i̶k̶e̶ — is pure text that copies and pastes into any platform.

Unlike markdown strikethrough (~~text~~) which only works on Discord and Reddit, Unicode strikethrough works everywhere — Instagram bios, WhatsApp messages, Twitter posts, email subjects, and gaming usernames.
