What is Morse Code?

Morse code is a method of encoding text characters as standardized sequences of dots (·) and dashes (−). Named after Samuel Morse, the co-inventor of the telegraph, it was widely used in early long-distance communication. Each letter, number, and punctuation mark is represented by a unique pattern.

How to Read Morse Code

  • Dot (·) — a short signal, the basic time unit
  • Dash (−) — a long signal, 3× the length of a dot
  • Space between letters — 3 dot-lengths (shown as a space)
  • Space between words — 7 dot-lengths (shown as /)

FAQ

Can I play Morse code as sound?
Yes! Click the Play button to hear your translated Morse code as audio beeps. Adjust speed, pitch, and volume using the sliders.
What does the Light button do?
When Light is enabled, the screen will flash white for dots and dashes, simulating a Morse code flashlight signal. Great for learning or signaling in the dark!
Does the Vibrate feature work?
Vibrate works on supported mobile devices. Your phone will vibrate in the pattern of the Morse code — short buzzes for dots, longer buzzes for dashes.
