Morse Code Translator

Translate text into Morse code, or decode Morse code back into English text. Copy and paste instantly.

What is Morse Code?

Morse code is a method used in telecommunication to encode text characters as standardized sequences of two different signal durations, called dots and dashes (or dits and dahs). It is named after Samuel Morse, an inventor of the telegraph.

How to read Morse Code?

Our translator uses the standard International Morse Code format:

  • Dot (.) is the basic unit of time measurement.
  • Dash (-) is three times the duration of a dot.
  • Space between parts of the same letter is equal to one dot.
  • Space between letters is equal to three dots (shown as a single space ' ').
  • Space between words is equal to seven dots (shown as a slash '/' in our translator).

Who uses Morse Code today?

While no longer the primary method for maritime or aviation communication, Morse code is still widely used by amateur radio operators (hams), in aviation navigation beacons (VORs), and as a popular puzzle or easter egg in video games, movies, and escape rooms.
