What is Morse Code?

Morse code is a method of encoding text characters as standardized sequences of dots (·) and dashes (−). Named after Samuel Morse, the co-inventor of the telegraph, it was widely used in early long-distance communication. Each letter, number, and punctuation mark is represented by a unique pattern.

How to Read Morse Code

  • Dot (·) — a short signal, the basic time unit
  • Dash (−) — a long signal, 3× the length of a dot
  • Space between letters — 3 dot-lengths (shown as a space)
  • Space between words — 7 dot-lengths (shown as /)
